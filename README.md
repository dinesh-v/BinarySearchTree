## Synopsis

Binary search tree implementation in Java 8

## Code Example

Initialize an undirected graph and add edges.

    BinarySearchTree b = new BinarySearchTree();
    b.insert(3);
    Node x = b.insert(8);
    b.insert(1);
    b.insert(4);
    b.insert(6);
    b.insert(2);
    b.insert(10);
    b.insert(9);
    
    (or)
    
    bst.insertWithArray(7, 5, 3, 6, 1, 4, 12, 9, 15, 8, 10, 13, 17);
